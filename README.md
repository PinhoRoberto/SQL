# SQL Scripts for Database Practice

This repository contains a collection of SQL scripts designed to support learning and practicing core database operations using MySQL. The scripts cover essential topics such as:

- Table creation and schema definition
- Data insertion and manipulation (INSERT, UPDATE, DELETE)
- Querying data with SELECT
- String and numeric functions
- UNION operations and joins
